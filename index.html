Authenticity Checks

1. QR code scans successfully in MediCheck
2. Holographic seal on packaging changes colour when tilted
3. NAFDAC registration number matches the label
4. Blister packs are intact with embossed manufacturer logo
5. Tablet/capsule colour, shape, and smell are consistent

Report if suspicious

If something looks wrong — different colour, texture, unusual smell, or QR code returns an error — do NOT consume the product. Report it immediately using the button below.
